##Install GNU Flex and GCC
```
  $ sudo apt-get install flex
  $ sudo apt-get install gcc
```
##How to run?
- flex foobar.flex and you'll get lex.yy.c
```
  $ flex foobar.flex
```
- Compile lex.yy.c using gcc
```
  $ gcc lex.yy.c -lfl
```
- then run 'a.out' with test.c as input
```
  $ ./a.out < test.c
```
